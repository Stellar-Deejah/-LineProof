# LineProof FAQ

## What is LineProof for?
High-demand services that need provably fair waiting lists: tickets, government appointments, healthcare, sneaker drops, scholarships, and housing lotteries.

## Does it prevent scalping?
Yes. Positions are non-transferable by design, so resale on secondary markets is not possible.

## Is user data private?
The protocol enforces eligibility on-chain without exposing personally identifiable information beyond the bound Stellar identity.

## How do I integrate it?
Follow [docs/integration-guide.md](integration-guide.md) and start with the TypeScript SDK.
